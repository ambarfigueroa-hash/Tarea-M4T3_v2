# YOLOv8 Construction Equipment Detection

## Problem
Detect construction equipment using computer vision.

## Model
YOLOv8s trained using Roboflow dataset.

## Metrics
Precision: 0.671  
Recall: 0.246  
mAP50: 0.376  

## Dataset
Roboflow project: Tarea-M4T3_v2

## How to Reproduce
1. Open notebook in Colab
2. Install ultralytics
3. Download dataset
4. Run training or inference

## Results
Training curves and predictions available in results folder.
